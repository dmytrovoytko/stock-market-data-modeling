# Stock Market Analytics/Crypto Machine Learning project 


## Run the project locally

1. Install packages
`pip install --no-cache-dir -r requirements.txt`

2. Run the workflow
`bash start_app.sh`
which sets environment and executes `wf_orchestrate.py`

## Run the project in Docker

`bash run-docker.sh`
